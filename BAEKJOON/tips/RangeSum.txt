### 구간 합
합 배열을 이용해 시간 복잡도를 더 줄이기 위해 사용하는 특수한 목적의 알고리즘

### 합 배열
기존의 배열을 전처리한 배열
시간 복잡도 O(N) -> O(1) 로 감소

배열 A가 있을 때 합 배열 S는 
S[i] = A[0] + A[1] + A[2] + ... + A[i - 1] + A[i]    // A[0] ~ A[i]까지의 합


### 합 배열 S를 만드는 공식
S[i] = S[i - 1] + A[i]

### i에서 j까지 구간 합을 구하는 공식
S[j] - S[i - 1]

### e.g., A[2] ~ A[5] 구간 합을 합 배열로 구하는 과정
S[5] = A[0] + A[1] + A[2] + A[3] + A[4] + A[5]
S[1] = A[0] + A[1]
S[5] - S[1] = A[2] + A[3] + A[4] + A[5]
