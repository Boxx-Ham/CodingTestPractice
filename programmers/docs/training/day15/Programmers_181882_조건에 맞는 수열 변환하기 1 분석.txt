###조건에 맞는 수열 변환하기 1 문제 분석

변환하는 조건

1. 50 이상의 짝수 -> 2로 나누기
2. 50 미만의 홀수 -> 2로 곱하기

arr를 쭉 돌면서
50 이상인지 아닌지 체크

1번 조건이면 2로 나누기
2번 조건이면 2로 곱하기

--------------------------------------------------------
### 수도 코드

결과값을 담을 result 배열을 arr의 크기로 만들기

for (i -> 0 ~ arr의 크기) {
    if (arr[i]가 50 이상의 짝수) {
        result[i]에 arr[i] / 2 대입
    } else if (arr[i]가 50 미만의 홀수) {
        result[i]에 arr[i] * 2 대입
    } else { 
        result[i]에 arr[i] 대입
    }
} 

result 배열 반환
