### 프로그래머스 181874 문제 : 특정한 문자를 대문자로 바꾸기

영소문자로 이루어진 문자열 my_string과 영소문자 1글자로 이루어진 문자열 alp가 매개변수로 주어질 때,
my_string에서 alp에 해당하는 모든 글자를 대문자로 바꾼 문자열을 return하는 solution 함수를 작성해 주세요.

## 제약 조건

1 ≤ my_string의 길이 ≤ 1,000

## 입출력 예시 (my_string / alp / result)
|---------------------------------------------------|
|   "programmers"   |   "p"     |   "Programmers"   |
|---------------------------------------------------|
|   "lowercase"     |   "x"     |   "lowercase"     |
|---------------------------------------------------|

-------------------------------------------------------------------

### 문제 분석

유니코드로 대소문자 변환할 때 32 만큼 차이가 남.
a -> A 로 변환 시 a - 32 해야 함.
A -> a 로 변환 시 a + 32 해야 함.

replace를 사용하면 될 듯 함
인자는 char로 넣어야 하니깐
alp를 char로 변환하여 replace 하면 될 듯

alp는 한글자로 이루어진 문자열이라고 했으니.

-------------------------------------------------------------------

## 수정사항

replace가 string string도 변환되는 걸 몰랐음.
그래서 char char로 변환해서 사용했는데
alp가 string이고 alp.toUpperCase()도 string 이므로
alp를 alp.toUpperCase()로 대문자로 변환하여 반환하면 더 쉽게 해결 가능

-------------------------------------------------------------------

### 수도 코드

my_string.replace(alp, (char) (alp - 32)) 반환
